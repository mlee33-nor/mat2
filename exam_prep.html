6.1 · Exponential Functions Concept Sheet

Definition. An exponential function has the form $f(x)=b^x$ where $b>0$, $b\ne 1$, $x\in\mathbb{R}$.

  • For each real $x$, $b^x$ is a unique real number.
  • All rational-exponent rules extend to irrational exponents.
  • If $b^a = b^c$ then $a=c$ (one-to-one).
  • If $b>1$ the function grows; if $0decays.

6.2 · Transformations & $e^x$ Concept Sheet

The number $e\approx 2.71828\ldots$ Because $2

Transformations of $y=b^x$

  • $y=b^{x+c}$ — shift left $c$ units (argument change).
  • $y=b^{x-c}$ — shift right $c$ units.
  • $y=b^x+k$ — shift up $k$; new asymptote $y=k$.
  • $y=b^x-k$ — shift down $k$; new asymptote $y=-k$.
  • $y=-b^x$ — reflect over x-axis; range becomes $(-\infty,0)$.
  • $y=b^{-x}$ — reflect over y-axis (growth ↔ decay).

Finding features after a transformation of $y=b^x+k$-style:

  • Asymptote: whatever the vertical shift is.
  • y-intercept: plug $x=0$.
  • x-intercept: set $y=0$ and solve (may not exist).
  • Domain: still $(-\infty,\infty)$.
  • Range: shift $(0,\infty)$ by the same vertical shift / reflection.

One-to-one: $y=b^x$ is one-to-one (horizontal line test) — so it has an inverse, which is the logarithm.

6.3 · Logarithms Concept Sheet

Definition. For $b>0,\;b\ne 1$: $$x=b^y \;\Longleftrightarrow\; y=\log_b x$$ Read: "$y$ is the exponent on base $b$ that yields $x$."

Mantra: a logarithm is an exponent.

Special notations

  • $\log x$ means $\log_{10} x$ (common log).
  • $\ln x$ means $\log_e x$ (natural log).

Cancellation / Inverse identities

  • $\log_b b^x = x$   (log of base-to-a-power returns the exponent)
  • $b^{\log_b x}=x$
  • Special cases: $\log 10^x = x$,   $\ln e^x = x$,   $10^{\log x}=x$,   $e^{\ln x}=x$.

6.4 · Logarithmic Graphs Concept Sheet

Rules for $y=\log_b(\text{argument})$

  • Domain: set argument $>0$ and solve.
  • Vertical asymptote: set argument $=0$ and solve.
  • x-intercept: set $y=0$, i.e. argument $=1$.
  • The larger $b$, the closer the graph hugs the x-axis.
  • Range of ALL basic log graphs is $(-\infty,\infty)$.

Building an x-y chart

Choose $x$ so the argument is a power of the base — then $\log_b b^n = n$.

Example values for $y=\log_2 x$: pick $x=\tfrac14,\tfrac12,1,2,4,8$ → $y=-2,-1,0,1,2,3$.
